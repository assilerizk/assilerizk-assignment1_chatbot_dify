Develop three chatbots using the Dify platform:
	- CB1: A chatbot that uses a website as its primary data source.
	- CB2: A chatbot that uses a collection of documents/files as its dataset.
	- CB3: A chatbot that uses a database schema as its dataset and connects directly to an MS SQL Server database.
